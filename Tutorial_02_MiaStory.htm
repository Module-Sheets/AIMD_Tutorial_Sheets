Tutorial 02 01 – Set Up Story Tracks

https://youtu.be/ZWk8-5G0P-o

  1. SELECT 24 FPS – IMPORTANT TO BE CONSISTENT DURING PROJECT!
  2. Bring in characterised Mia from Resources/Asset Browser/Tutorials.
  3. De-characterise, add Floor Plane (like Tutorial 01).
  4. PLOT TO CONTROL RIG.
  5. Test Floor Plane by moving up.
  6. In Story create Character track, select Mia in dropdown box.
  7. Right click in transport area of Character Track and insert Animation File from provided files.
  8. Select “01 Clip Stand Loop”, drag to 0 (start) and 19 (end).
  9. Enter 19 in end box of transport control window (and timeline).
  10. Hit F to frame selected animation clip in Story.
  11. Check loop at start/end by FF and Rewind.
  12. Turn loop on in Transport control.
  13. Select root between feet in Character Controls window.
  14. Click Ghost box in Story Track (Eye), root, and clip (all selected). Note the Ghost Skeleton.
  15. Press T for translation gizmo.
  16. Ensure translation is zeroed.
  17. Press R and check rotation zeroed.
  18. Character should be centered within root and on floor – check.
  19. Right click in left panel of Story, insert new Character Track, select Mia.
  20. This provides a separate track and overrides the one above.
  21. Right click, insert anim file – it comes in at current transport position.
  22. Select “Clip Stand Talk 1a Poses.fbx”, drag clip to 0 (start).
  23. With clip selected, press F to frame.
  24. Adjust transport to 0 (start) and 200 (end).
  25. Repeat steps 6-10 to check.
  26. Go to 0 (start), mute story track for “Clip Stand Talk 1a Poses.fbx”.
  27. Check there is no jump in stand pose.
  28. Rename both Story Character tracks as “Stand Looped” and “Stand Talk 1a Looped Poses”.

Tutorial 02 03 – Process Tracks from Story

https://youtu.be/ME5JhHQgfM0

  1. Click track with subtrack pose, make sure it is active.
  2. Right click on track name box and select “process track and subtracks to new clip”.
  3. Check fps at 24 and accept all defaults.
  4. Save clip as “Clip Stand Talk 1a Looped”.
  5. The FBX animation-only clip is saved to disk and in new track.
  6. Repeat the process for clips 1b and 1c.
  7. Try copying and pasting the clips into one timeline for export of all animations.
  8. To export animations in-scene, plot clips to takes on the skeleton; try on a new take and save.

Recap – Two Ways to Add Animations

  1. Week 1 Method – Merge whole FBX files to scene in Live Viewing Pane, or Motion File Import to FBX Skeletons
    1. Merge whole FBX, characterise, drive character FBX, plot to Control Rig, insert in Story.
    2. Motion File Import with FBX Skeleton selected – additional takes added.
    3. Insert current takes into Story track one by one.
    4. Can also insert anim files from whole FBX files – will do with Mixamo.
  2. Week 2 – Import FBX anim-only clips in Story Mode
    1. Import FBX animation-only clip into Story track of character FBX with Control Rig.
    2. Add more clips on same track or additional tracks for editing.
    3. Paste together in one track for export.
    4. Can also insert anim files from whole FBX files – will do with Mixamo.

Tips to Remember

  1. CHECK FPS!!!
    1. Week 1 – 30 fps
    2. Week 2 – 24 fps
  2. Always characterise on T-pose.
  3. Plot to Control Rig before going to Story.
    1. Plot rather than select Control Rig to keep animation.

Story

  1. Need FBX with Control Rig in Story for full effect.
  2. Check Ghost with root, clip, Ghost button selected.
  3. Zero out Ghost Translation and Rotation (ONLY on ROOT).
  4. Insert New Character Animation Track.
  5. Insert Current Take.
  6. Export clips (animation only!)
    1. Right click and make clip writeable.
    2. Check clean frames (no *).
    3. Change transport end frame to match.
    4. Keys go one frame beyond – remember when exporting.
    5. Export ends at transport end – set appropriately.
    6. Right click to save clip; label animation-only clips as Clip_Name.
    7. Or process track/subtrack to a new track to export (clips).
    8. Transport at zero.
  7. Right click to import clips
    1. Insert animation file.
    2. Imports at current transport position.
  8. TURN KEY FRAMING OFF AFTER USING.
  9. Mute Story to go back to Live view and process more clips.

Poses

  1. Copy and paste pose on:
    1. Full body mode.
    2. Full selection WITH root.
    3. Keep pose defaults.
