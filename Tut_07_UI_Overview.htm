7.2 Text Readout Score (Extra)
  1. Create a UI Text object and name it Score Counter.
  2. Add a script component.
  3. Create variables:
    • private Text uiText;
    • private long score = 0;
  4. In Start(), cache uiText and call updateScoreCounter().
  5. Create addPoints(int points) to increment score and refresh the text.
  6. Implement updateScoreCounter() outside Update().

Example output format: <color=blue>Score</color>: 0

7.3 Animate UI Button (Legacy, Basic - Extra)
  1. Create a button from UI > Legacy > Button.
  2. Keep Canvas set to Screen Space - Overlay.
  3. Set button Transition to Animation.
  4. Create and assign an Animator Controller.
  5. Create additional clips: Highlighted, Pressed, Selected.
  6. Set up Animator transitions and trigger parameters.
  7. In Animation window, add properties such as RectTransform scale/rotation and Image color.
  8. Test edits by scrubbing timeline, adding keys, and toggling record mode.

7.4 Animate UI Button with TextMeshPro (TMP - Extra)

Video reference

  1. Create a button from UI > Button - TextMeshPro.
  2. Observe TMP button structure on Canvas (button + nested TMP text).
  3. Compare legacy vs TMP button workflows.
  4. Install TMP essentials when prompted (first-time TMP setup).
  5. Import examples/extras and fonts from Blackboard resources.
  6. Use Window > TextMeshPro > Font Asset Creator:
    • Drag font into Source Font File
    • Generate Font Atlas
    • Save font asset (ideally in a dedicated font folder)
  7. Assign font asset to button text and adjust size/color.
  8. Adjust button size, position, color, sprite and borders as needed.
  9. Switch transition from Color Tint to Animation and auto-generate animation setup.
  10. For clip tuning, disable Loop Time where useful and test behavior.
  11. Advanced: animate button/text properties, including scale, color, slight pressed rotation, and sprite variations.
