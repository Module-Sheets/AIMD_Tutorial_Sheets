    Tutorial 8 04 – VS – iPad Diner UI

    1. On iPad Canvas, Create a Button to Spawn a Diner - move this to the side of the existing timer from above
    2. change the text below the button object to “Diner 0”
    3. On iPad Canvas, Create a Visual Script Machine component and select Embed. We will be making the following Scene Variables and Visual Script:

    https://gyazo.com/3f44e72da9f0a1d3e346d19cf0bc094d

    https://gyazo.com/534b30098f891285465053c4763b5ecc

     

    1. Create Scene Variables of the following type and name:
      1. List of Game Object: DinerList (make 4 entries, but leave blank)
      2. Integer: DinerListIndex (leave as 0)
      3. List of String: OrderList (make 3 entries, but leave blank – to be used later)
      4. Boolean: OrderBool (Leave unticked)
      5. GameObject: MainCamera (Drag in Main Camera here)
    2. We will use the above in creating and managing a List of Diner Prefabs
    3. In the Visual Scripting Graph Edit window, create an On Button Click node, and Drag in the Diner 0 button above
    4. Get a Set Variable Node, select DinerListIndex in the drop down box
    5. get an Integer Literal Unit Node, type in 0, and wire up this value to the bottom link in the Set Variable Node
    6. We will Instantiate the Diner prefab now: Get a Game Object Instantiate Node, and drag the Diner prefab from the Project window into the box
    7. For position and rotation, place an empty game object on the floor below the chair to spawn onto
    8. Get a Game Object Node in the VS Edit Graph Window, select the Empty Object for Pos and Rot
    9. Right click in the VS Script Graph edit window and get Transform:Get Position and Transform:Get Rotation Nodes
    10. Plug the Empty object Game Object Node into each of these inputs
    11. Wire the outputs to the Game Object Instantiate Node
    12. Test it now.
    13. If working, finish up by getting a Node for List: Insert Item
    14. plug in the green continue arrow from Instantiate, and the Game Object to the Item slot in The List Insert Item Node
    15. Make a new Get Variable Node, and select the Diner List Variable from the drop down box. Plug this into the List slot in the List Insert Item Node
    16. You can add an optional Debug log Node and string after this to check success.
