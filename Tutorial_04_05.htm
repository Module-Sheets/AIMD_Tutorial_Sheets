Tutorial 04 00 - Mia Animator Recap

https://www.youtube.com/watch?v=miEewb5lXwo

  1. Ensure all example Models and clips have same Humanoid Rig and Avatar from Mia or Mixamo TPose File
  2. Ensure Clips Stand Loop (Idle) and Stand Talk 1 a,b,c are rooted with no compression (either root or no root methods)
  3. Take Loop off Stand Talk 1a, b, c
  4. Drag Stand Loop into Animator Window - Right click and Set as Layer Default State
  5. This runs the idle animation off the Entry Node
  6. Drag the Clips Talk 1 a,b,c into the Animator Window
  7. Right Click on the Any State Node and make transitions to each in turn
  8. Right Click on each of the 3 Talk clips and transition each to the Exit node
  9. Click on Animator window Parameters Tab
  10. Create Three new Trigger Parameters and name Talk 1a, Talk 1b, Talk 1c
  11. Right click transitions from Any State and add a condition using the appropriately named Trigger
  12. Make sure Has Exit Time is Unchecked here
  13. Make sure Has Exit Time is Checked in transitions to the Exit Node

Tutorial 04 02 / 05 - Follow-up Workflow and Dialog Integration

https://www.youtube.com/watch?v=iCLnz5oUIpU

1. Canvas:

  1. Create UI Canvas with right click in Hierarchy
  2. Change name to Canvas Mia Dialogue
  3. Change Render Mode in Inspector to World Space
  4. Drag in Main Camera to Event Camera (handles click events)
  5. Rect Transform
    1. Zero out position
    2. Make position -.6, 1.5 0 or next to Mia's head
    3. Make Width 600, Height 300
    4. Make Scale .001 for X,Y,Z (1 Unit = 1 M!)
    5. Rotate Y to 180

https://gyazo.com/84ca6a68a0b8c75c032fd6fa5b5ff8d5

2. Text:

  1. Right click Canvas in Hierarchy and create UI/Text

(Legacy/Text or Text Mesh Pro. if text mesh Pro, accept all imports of essentials and examples)

  1. Name Text As Dialogue Text 0
  2. Change Width to 500 and Height to 60
  3. Change Position to -15, 88, 0 or at top of canvas
  4. Change font size to 28

3. Visual Script Graph:

  1. Use previous Script Graph on the Mia character with the Debug function
  2. Add Text Set Text node
  3. Drag Dialog text into game object slot
  4. Add new String Literal node
  5. Type Mia: Hello, can you take my order? in the box
  6. Wire as below and play test (NOTE: May need to use ON MOUSE DOWN, not ON POINTER DOWN)

https://gyazo.com/8387b05762be58a183d8d3b79d61c6bc

If you are using Text Mesh Pro, use Text Mesh Pro UGUI: Set Text with Argument. USE ON MOUSE DOWN, not ON Pointer Down.

https://gyazo.com/bfe76bb5ea260a493c90d0a862bb3a71

4. Dialogue system - Dialogue Counter, Dialogue list, Button

  1. Create a Visual Script as before (Embed) on Canvas Dialogue
  2. Create Scene Variable in Graph Inspector:
    1. Dialogue List - Type: List of String
    2. Anim Trigger List - Type: List of String
    3. Dialogue Counter - Type: Integer (this is incremented to access next items in the animation and dialogue lists)
  3. Create a UI button on dialogue canvas for forwarding
    1. Position at end of Dialogue Text Box
  1. UI Button canvas will need ForceCanvasSortOrder.cs script to function

Rough tasks:

  1. Script Graph on Dialogue Canvas - Embed to access scene variables
  2. Create Scene Variables: Type List of String for Anim Triggers and Dialogue Text
  3. Create Scene Variable: Type Integer for Dialogue Counter
  4. Add nodes to set dialogue variable and increment with a button or key input
  5. Use dialogue counter to access Arrays incrementally to trigger animations and set Unity UI dialogue text
  6. Organise with Ctrl-LMB click drag to make housekeeping boxes

NOTE: Changing Scene Variables in the Script graph will probably require a refresh - drag the scene variables back into graph window to replace and refresh them.
