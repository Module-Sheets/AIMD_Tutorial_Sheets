Tutorial 02 03 - Process Tracks from Story
https://youtu.be/ME5JhHQgfM0

 

  1. Click track with subtrack pose, make sure is active
  2. Right click on track name box and select “process track and subtracks to new clip”
  3. Check fps at 24 and accept all defaults
  4. You will be prompted to save a clip. Save this in the format “Clip Stand Talk 1a Looped”
  5. The Fbx animation only clip is saved to disk and in new track
  6. Repeat the process for clips 1b and 1c
  7. Try copying and pasting the clips into one timeline for export of all animations.
  8. Exporting these gives you clips. If you want the animations in the scene for export, you would have to plot the clips to takes on the skeleton. Try this on a new take and save the scene.

 

Note: Try Opening an exported Clip on its own and check the difference….

 

Recap - Two ways to Add animations :

  1. Week 1 Method – Merge Whole FBX files to scene in Live Viewing Pane, or Motion File Import to FBX Skeletons
    1. Merge whole fbx, char, use to drive char fbx, plot to cr, insert in story
    2. Motion file Import with FBX Skeleton Selected – additional takes added
    3. Insert current takes into story track one by one
    4. Can also insert anim files from whole fbx files – will do with Mixamo
  2. Week 2 – Import FBX anim only clips in Story Mode
    1. Import fbx only animation clip into story track of char fbx with CR
    2. More clips by right click on same track or additional with same char for editing
    3. Paste together in one track for export
    4. Can also insert anim files from whole fbx files – will do with Mixamo

Tips to Remember:

  1. CHECK FPS!!!
    1. Week 1 – 30 fps
    2. Week 2 – 24 fps
  2. Always Characterise on Tpose
  3. Plot to Control Rig before going to Story
    1. Plot rather than select Control Rig to have animation

Story

1.       Need Fbx with control rig in story for full effect

2.       Check Ghost with root, Clip, Ghost button selected

3.       Zero out ghost Translation and Rotation (ONLY on ROOT)

4.       Insert New Character Animation Track

5.       Insert Current Take

6.       Export clips (animation only!)

a.       Right click and make clip writeable

b.       Check Clean Frames - Frames with no *

c.       Change transport end frame to match

d.       Keys go one frame beyond – remember when exporting

e.       Export Ends at transport end – be careful to change as needed

f.        Right click to save clip and Label animation only clips as Clip_Name…

g.       or Process track/subtrack adj to new track to export (clips)

h.       Transport at zero

7.       Right click to import clips

a.       Insert animation file

b.       Imports on current transport position

8.       TURN KEY FRAMING OFF AFTER USING

9.       Mute story to go back to Live view and process more clips in (tutorial 1)
