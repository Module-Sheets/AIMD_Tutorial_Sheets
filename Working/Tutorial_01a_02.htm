Tutorial 01.01 – Getting Started with Characterised FBX Skeletons and Mocap

Video Tutorial: FBX Skeletons and Mocap

Mocap Files:
Tutorial Files 01 Mocap Only.zip

Initial Setup Steps

  1. Open MotionBuilder and start a new scene from the top left File Menu
  2. Arrange your layout to suit with Navigator, Resources, Viewer, and Character Controls Windows Only – REQUIRES YOU TO HOLD DOWN SHIFT
    1. Drag Resources top right
    2. Drag Navigator top right below resources
    3. Close animation layers
    4. Close animation key controls
    5. Keep timeline at bottom across whole UI
    6. Keep Character Controls full right

Character Template & Mocap Setup

  1. From Resources/Asset Browser Tab/Tutorials Folder/, Drag a Mia Blue into the scene (FBX Open/No Animation)
  2. From Resources/Asset Browser Tab/Templates Folder/Characters Folder/, Drag a Character Template over any part of Mia and drop
  3. Select Characterise in the pop-up
  4. Check Mia is facing positive Z and click Biped
  5. Find in navigator under Characters and rename to Character Mia
  6. At top menu on Left, select File/Merge
  7. Select Waiter Idle.fbx from mocap files above
  8. Accept all Defaults and click Merge
  9. Select Takes in dropdown menu in Transport Controls Window to see new and old takes (Take 001 and Waiter Idle)
  10. In navigator expand the Scene Folder by clicking the "+" sign
  11. Find, select, and delete the Cameras, Markers, and unnamed markers in Scene Folder
    1. Right Click on System Folder and click "Select Branches"
    2. Right Click, Select Delete/Confirm
    3. Right Click on "Unnamed Markers" and click "Select Branches"
    4. Right Click, Select Delete/Confirm
  12. You should be left with the mocap FBX skeleton and named optical markers
    1. Select Display/XRay at top of Viewer to view bones
  13. Check framerate at bottom far right of Transport Controls (select 30 in this case)
  14. (Frame 0 – type 0 into start frame to eliminate 0* - Important)
  15. Also change 894* to 894
  16. Avoid moving the blue spheres that adjust the transport window – this shrinks the transport view

Characterise Mocap Skeleton

  1. Characterise mocap FBX Skeleton on T-pose at Frame 0
  2. Rename Character as Character Mocap
  3. In Character window, with Character Mia as Character, select Character Mocap as Source
  4. Playback and check the mocap Drives Mia, Note the scale differences

Tutorial 01.02 – Adjustments, Floor Plane & Control Rig

Video Tutorial: Adjustments and Floor Plane

Character Settings & IK Blend

  1. Look at Property tabs in Navigator with Characters/Character Mia selected
  2. Select Character in Navigator window, look at Property tabs for Character Definition, Character Settings
  3. Character Settings Tab – Look at Inputs – Check Stance and Active tickbox (for all changes we make!)
  4. Most of the Same controls are duplicated in the Character Controls Window we just used
  5. Adjust IK Blend (Reach) and Pull – IK Blend moves toward source, Pull brings the rest along – Example: IK Blend Example
  6. IK (Inverse Kinematics) is animation propagating up a skeleton (wrist moves elbow), FK (Forward Kinematics) is just the individual bone rotating
  7. Adjust head IK Blend, T, and R, then IK Pull when IK Blend is active
  8. In Character Settings Tab, try match source and action space compensation
  9. Adjust Hands and other body parts for IK Blend T,R and Pull in Character Settings and also Character Controls windows. See the correlation and effect
  10. Check feet – they are below the floor
  11. In Character Settings tab, make sure character is on stance (not T-pose!), then untick active box
  12. In Character Definition tab, untick characterize box
  13. Check Character Controls Window for green bones (correct stance)

Floor Plane Setup

  1. Drag in Plane object from Asset Browser/Templates/Elements/
  2. Type 0, 0, 0 in bottom right of viewing pane
  3. Alt-drag the plane into the four spots below the reference node mapping box in Character Definition
  4. Tick Characterize button, accept Biped
  5. In Character Settings, modifiers/floor contact, select always sync in drop down box
  6. Tick feet floor contact and toes floor contact
  7. Scale up feet contact size to 300
  8. Change foot front to 8
  9. Change foot outside to 8
  10. Click active box while in stance input
  11. Change source to character mocap
  12. Check foot plane works by hitting t key and translating plane up and down

Tutorial 01.03 – Importing More Mocap & Adjusting with Control Rig in Story

Video Tutorial: Control Rig in Story Mode

Control Rig Setup & Story Mode

  1. Clean up scene in navigator to just have FBX
  2. Select FBX Solving Skeleton in Navigator and right click select branches
  3. At top left select Motion File Import and import with Merge/In Selected Models only and all tick boxes selected except for create unmatched models
  4. Change start frame from 0* to 0 and also clean up end frame
  5. Check fps = 30
  6. In Takes drop-down box in Transport bar, select and check different takes of motion
  7. In Story Mode, insert Character Animation Track and select Character Mia in Drop down box reference
  8. In Character Settings, Plot Character to Control Rig for Adjustments in Story
  9. Select FK/IK and 30 fps
  10. Name Control Rig as Control Rig Mia
  11. Try IK Blend and Pull to move skeleton – Example: IK Blend Example
  12. In story right click in story track and insert current take
  13. Story Overrides all live animation! Control rig is plotted
  14. Can mute story and change source to go back to mocap
  15. Can switch takes, plot back to same control rig, add to second Story track
    1. Plot will capture from start and end frames of transport control!!!
    2. Can move animations
    3. Plot only between the start and end frames of takes – check!
    4. Plots to transport position – keep at 0 when plotting
    5. Plot to different tracks to adjust individually
  16. Try back and forth with a few animations. Try to catch any pitfalls
