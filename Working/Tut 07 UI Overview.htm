2.       7.2 Text Readout Score (extra)

a.       Make UI/Text – call it “Score Counter”

b.       Add Component/Script

c.       Create Text “uiText” and “score” variables

private Text uiText;
                private long score = 0;

d.       In Start – uiText = GetComponent<Text>();

void Start () { 
                                uiText = this.GetComponent<Text> ();
                                updateScoreCounter ();
                } 

e.       Add Points (run script when key is pressed - remember how to do this?)
                public void addPoints(int points){
                                score += points;
                                updateScoreCounter ();
                }

f.        Method outside of update:

private void updateScoreCounter(){
                                uiText.text = "<color=blue>Score</color>: " + score;
                }
